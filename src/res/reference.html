  • Голосовой помощник может открыть карту в браузере Chrome и проложить маршрут с учётом пробок на дорогах.
    По умолчанию, используется проезд на общественном транспорте.

    Произнесите голосовую команду: Построй маршрут от [начальная точка] до [точка назначения].

    По-возможности, произносите адреса подробно. Пример: "Построй маршрут от Москва Тверская 5 до Москва Тверская 7".

  • Голосовой помощник может открыть сайта сервиса яндекс.погода в браузере Chrome с прогнозом на день текущей даты.

    Произнесите голосовую команду: Покажи погоду в городе [название города в именительном падеже].

    Пример: "Покажи погоду в городе Санкт-Петербург".

  • Голосовой помощник может выполнить поиск файла только по имени на рабочем столе компьютера и открыть проводник с выделенным найденным файлом. На данный момент возможен поиск файлов только с наименованием на русском языке.

    Произнесите голосовую команду: Найди файл на рабочем столе по имени [название файла на русском без расширения].

    Пример: "Найди файл на рабочем столе по имени резюме".

  • Голосовой помощник может открыть файл, ранее найденный с помощью голосовой команды, указанной выше в руководстве. Открытие производится программой по умолчанию для данного типа файла.

    Произнесите голосовую команду: Открой файл.

  • Голосовой помощник может закрыть активное окно, (например, файла или проводника), с которым в данный момент работает пользователь.

    Произнесите голосовую команду:
    Закрой окно.
    Закрой файл.

  • Голосовой помощник может свернуть активное окно, (например, файла или проводника), с которым в данный момент работает пользователь.

    Произнесите голосовую команду:
    Сверни окно.
    Окно сверни.

  • Голосовой помощник может выполнить поиск файлов определённого типа на рабочем столе. Например, найти все файлы расширения jpeg или png (все фотографии), или расширения docx и pdf (все документы), или расширения mp4 и MOV (все видео) и отобразить их в проводнике.

    Произнесите одну из голосовых команд:
    Покажи в проводнике [все фотографии] с рабочего стола.
    Покажи в проводнике [все документы] с рабочего стола.
    Покажи в проводнике [все видео] с рабочего стола.

  • Голосовой помощник может открыть в браузере Chrome следующие сайты: google.com, yandex.ru, youtube.com, также может открыть google-диск и google-переводчик.

    Произнесите одну из следующих голосовых команд: Открой [открой поиск гугл],
    Открой [открой яндекс],
    Открой [открой google диск],
    Открой [открой google переводчик],
    Открой [открой youtube],

  • Голосовой помощник может открыть в браузере Chrome поисковые системы: google.com, yandex.ru, youtube.com , и выполнить в них поиск по голосовой команде.

    Произнесите одну из следующих голосовых команд:
    Поиск (или найди) в google [поисковой запрос]
    Поиск (или найди) в яндексе [поисковой запрос]
    Найди в youtube видео [поисковой запрос].

    Пример: "Найди в google фото птиц".

  • Голосовой помощник может закрыть активную вкладку в браузере Chrome по голосовой команде.

    Произнесите голосовую команду:
    Закрой активную вкладку.
